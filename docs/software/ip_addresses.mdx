It is common to group similar components in a range of static IP addresses, such as:

| IP minimum      | IP maximum      | Component                              |
| :-------------- | :-------------- | :------------------------------------- |
| 192.168.131.1   |                 | Main Computer                          |
| 192.168.131.2   |                 | Main Microcontroller                   |
| 192.168.131.3   | 192.168.131.9   | Additional Control Components          |
| 192.168.131.10  | 192.168.131.19  | Camera                                 |
| 192.168.131.20  | 192.168.131.29  | Lidar                                  |
| 192.168.131.30  | 192.168.131.34  | GPS                                    |
| 192.168.131.35  | 192.168.131.39  | Sonar                                  |
| 192.168.131.40  | 192.168.131.44  | Arm                                    |
| 192.168.131.45  | 192.168.131.49  | Gripper                                |
| 192.168.131.50  | 192.168.131.69  | Access Points and Networking Equipment |
| 192.168.131.70  | 192.168.131.99  | Unassigned for Future Use              |
| 192.168.131.100 | 192.168.131.254 | Open DHCP Addresses                    |
