| Symbol | Symbol Name         |  Alt Code  | Purpose                                                  | Correct Example               |
| :----: | :------------------ | :--------: | :------------------------------------------------------- | :---------------------------- |
|   ±    | Plus-Minus Sign     | ALT + 241  | Symmetric tolerance                                      | 3.4±0.2                       |
|   °    | Degree Sign         | ALT + 0176 | Used in dimensions, temperature, angle                   | 90°                           |
|   ·    | Interpunct          | ALT + 0183 | Separate elements of dimensions                          | N·m                           |
|   µ    | Mu                  | ALT + 0181 | mirco, used in dimensions                                | µm                            |
|   -    | Hyphen              |            | compound words                                           | long-term                     |
|   -    | Hyphen              |            | Non-SI thread callout                                    | #0.25-20                      |
|   -    | Hyphen              |            | Dimension range                                          | Ø3.2 - Ø3.6                   |
|   —    | em-dash             | ALT + 0151 | Separate the two parts of an Item Name                   | Screw, Socket Head—M3×0.5 X 8 |
|   –    | en-dash             | ALT + 0150 | Not used in Item naming. Literary use is for item ranges |                               |
|   X    | Capital X           |            | Quantity                                                 | 3X screws                     |
|   X    | Capital X           |            | By, dimensions                                           | 2 X 4 lumber                  |
|   ×    | Multiplication Sign | ALT + 0215 | Thread Pitch                                             | M3×0.5                        |
|   #    | Hash Sign           |            | Non-SI dimension                                         | #0.25-20                      |
|   M    | Capital M           |            | Metric thread callout                                    | M3×0.5                        |
|   Ø    | Slashed O           | ALT + 0216 | Indicate diameter                                        | Ø3.4                          |
|   R    | Capital R           |            | Indicate radius                                          | R1.7                          |
