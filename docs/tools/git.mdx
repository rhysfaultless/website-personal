## Intallation Process

1.  [Download](https://git-scm.com/downloads) for your Operating System
2.  Follow the installation process from the download
3.  Open a terminal, and run:

    ```
    git config --global user.name "John Doe"
    git config --global user.email johndoe@example.com
    ```

    Changing the name values to your actual settings used on GitHub, GitLab, or Bitbucket.
    My settings are:

    ```
    git config --global user.name "Rhys Faultless"
    git config --global user.email rhysfaultless@gmail.com
    ```
