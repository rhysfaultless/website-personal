### Processes

<table>
  <tr>
    <th>Method</th>
    <th>Common Components and Uses</th>
    <th>Profile Tolerance <br /><em>my values in 2024</em></th>
  </tr>
  <tr>
    <td>3D printing, DMLS</td>
    <td>
      <ul>
        <li>Fluid paths and manifolds, where conventional manufacturing methods cannont produce the geometry.</li>
      </ul>
    </td>
    <td>±0.15 mm</td>
  </tr>
  <tr>
    <td>3D printing, FDM</td>
    <td>
      <ul>
        <li>Prototyping parts. It is unusual to use FDM for production parts.</li>
      </ul>
    </td>
    <td>±0.45 mm</td>
  </tr>
  <tr>
    <td>3D printing, SLS</td>
    <td>
      <ul>
        <li>Low volume plastic parts, where where tooling costs for vaccuum forming or injection moulding are acceptable.</li>
        <li>Complex geometry, that does not allow moulding or machining.</li>
        <li>Air ducts</li>
        <li>Camera mounting brackets (which often are mounded on complex angles)</li>
      </ul>
    </td>
    <td>±0.25 mm</td>
  </tr>
  <tr>
    <td>Forming, Sheet Metal</td>
    <td>
      <ul>
        <li>Chassis</li>
        <li>Body panels</li>
        <li>Brackets and structures for mounting components</li>
      </ul>
    </td>
    <td>±0.75 mm</td>
  </tr>
  <tr>
    <td>Injection Moulding</td>
    <td>
      <ul>
        <li>Aesthetic panels and covers</li>
      </ul>
    </td>
    <td>±0.15</td>
  </tr>
  <tr>
    <td>Laser Cutting</td>
    <td>
      <ul>
        <li>Sheet metal panels, covers, and chassis</li>
        <li>Plastic insulating covers, and terminal blocks</li>
      </ul>
    </td>
    <td>±0.25 mm</td>
  </tr>
  <tr>
    <td>Machining (Milling and Turning)</td>
    <td>
      <ul>
        <li>Drivetrain rotating components</li>
        <li>O-ring grooves</li>
      </ul>
    </td>
    <td>±0.15 mm</td>
  </tr>
  <tr>
    <td>Moulding, Carbon Fibre and Fibreglass</td>
    <td>
      <ul>
        <li>Aesthetic panels and covers</li>
        <li>Chassis, if sheet metal is not a viable option</li>
      </ul>
    </td>
    <td>Depends on process and material. Post moulding machining tolerances can be ±0.15 mm</td>
  </tr>
  <tr>
    <td>Waterjet Cutting</td>
    <td>
      <ul>
        <li>Glass lenses</li>
        <li>Foam and rubber gaskets</li>
        <li>Sheet metal</li>
      </ul>
    </td>
    <td>±0.15 mm</td>
  </tr>
