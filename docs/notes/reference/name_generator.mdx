## How this works

This page is loading a JavaScript component that renders a `<button>` and `<h2>` element.
The component imports a JSON file of words.
The component then chooses 2 words randomly, combines them, and then renders them to the `<h2>` element.
Whenever you click the `<button>`, the component gets 2 new random integers and updates the words based on those 2 index values.
